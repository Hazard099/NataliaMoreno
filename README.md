The codebase follows the standard HTML5 UP structure with minor modifications:

* **`index.html`**: The main page structure, including custom styles for the split hero and floating logo effect.
* **`assets/css/main.css`**: The core stylesheet, including the full reset, grid system, component styling, and custom navigation overrides.
* **`assets/js/`**: Contains core JavaScript files for templating effects (e.g., scrolling, preloading).
* **`images/`**: Contains all visual assets, including campaign logos and hero images.
